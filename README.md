# GroceryAPI

A simple api for manging your store's products.

### Prerequisites

Golang is required. Docker is (optional).

## Installing

### the prereqs
  https://go.dev/dl/
  
  https://www.docker.com/get-started/

### the code
    git clone git@github.com:ecarter202/groceryapi.git

## Running

### the tests
    go test

### the code
    cd bin
    go run main.go --debug
